## &#10162; Introduction:
Bootstrap is a popular front-end framework that provides a collection of pre-defined and reusable CSS and JavaScript components for building mobile-first responsive web pages. It offers a wide range of pre-defined elements for creating responsive layouts. And it is famous for CSS grid system and flexbox system to meet easiest way to implement them in a web page.

### &#9780; Overview:
1. [Downloading Bootstrap](#-downloading-bootstrap)
2. [Linking Bootstrap](#-linking-bootstrap)
3. [Structure of Bootstrap Page](#-structure-of-bootstrap-page)

### &#10022; Downloading Bootstrap:
 - To download Bootstrap from its official website: https://getbootstrap.com/
 - Once downloaded, It need to be extracted the contents of the ZIP file for your projects.
 - It is preferred for mostly development purpose and locally available projects.

### &#10022; Linking Bootstrap:
 - To use Bootstrap in your production web page, It need to be link the CSS and JavaScript files.
 - For links, refer official website.

### &#10022; Structure of Bootstrap Page:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Example</title>
  <link rel="stylesheet" href="{{..cdn..}}/bootstrap.min.css" media="all">
</head>
<body>
  <script src="{{..cdn..}}/bootstrap.bundle.min.js"></script>
</body>
</html>
```
